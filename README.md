# Queue (Q)

**Split the bill. Stay premium. Spend Less.**

## Features

-   **Landing Page**: Comprehensive introduction to the service highlighting value propositions, FAQs, and how it works.
-   **Claim Flow**: Secure flow for users to claim gifts and subscriptions effectively.
-   **Dashboard**: User dashboard to manage shared subscriptions.
-   **WhatsApp Integration**: Direct support and interaction via WhatsApp.
-   **Mobile First Design**: Fully responsive UI optimized for mobile devices.

## Tech Stack

-   **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
-   **Library**: [React 19](https://react.dev/)
-   **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your machine.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd q-website
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    # or
    bun install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    # or
    bun dev
    ```

    > **Note:** The development server is configured to run on port **3001** by default (`http://localhost:3001`).

4.  **Open in Browser:**
    Open [http://localhost:3001](http://localhost:3001) with your browser to see the application.

## Project Structure

-   `src/app`: Contains the App Router pages, layouts, and API routes.
-   `src/components`: Reusable UI components (Hero, Header, Modals, etc.).
-   `public`: Static assets including images and SVGs.
